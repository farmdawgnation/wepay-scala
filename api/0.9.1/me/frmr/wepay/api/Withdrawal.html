Withdrawal

case class Withdrawal(account_id: Long, amount: Option[Double], redirect_uri: Option[String], callback_uri: Option[String], note: Option[String], withdrawal_id: Option[Long], state: Option[String], withdrawal_uri: Option[String], recipient_confirmed: Option[Boolean], create_time: Option[Long]) extends ImmutableWePayResource[Withdrawal, WithdrawalResponse] with Product with Serializable

Represents an instance of a Withdrawal.

account_id

The WePay Account ID associated with the Withdrawal.

amount

The withdrawal amount.

redirect_uri

The URI to redirect users to when finishing the Withdraw flow.

callback_uri

The URI for WePay to send IPN messages to.

note

The note to be attached to the withdrawal.

withdrawal_id

The ID of the Withdrawal on WePay's system. This should only be populated by WePay.

state

The current state of the Withdrawal.

withdrawal_uri

URI to view the Withdrawal on WePay's site.

recipient_confirmed

Actually, I'm not entirely sure what this represents. Ooops.

create_time

The time of creation, ya dig?
