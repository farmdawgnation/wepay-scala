ImmutableWePayResourceMeta

trait ImmutableWePayResourceMeta[Model <: ImmutableWePayResource[Model, CrudResponseType], CrudResponseType] extends WePayResourceMeta[Model]

Children of this trait are meta objects used to find or save resource instances that cannot be modified after their creation.

This class of resources include Checkout, Preapproval, and Withdrawal. Additionally, this Meta object also introduced the CrudResponseType for operations such as save that return JSON of a different type than those that return an instance of the object.
