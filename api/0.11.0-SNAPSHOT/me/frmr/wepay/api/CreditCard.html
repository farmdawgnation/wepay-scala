case class CreditCard(userName: String, email: String, ccNumber: Option[String] = scala.None, cvv: Option[Int] = scala.None, expirationMonth: Option[Int] = scala.None, expirationYear: Option[Int] = scala.None, address: Option[CreditCardAddress] = scala.None, creditCardId: Option[Long] = scala.None, creditCardName: Option[String] = scala.None, state: Option[String] = scala.None, referenceId: Option[String] = scala.None) extends ImmutableWePayResource[CreditCard, CreditCardResponse] with Product with Serializable

An instance of a Credit Card on WePay.

This class is unique because a handful of the values that are required for saving the credit card will not be returned when you retrieve it later. I've annotated those values in the description below. As always the most official docs are WePay's.

userName

The name of the cardholder.

email

The email of the cardholder.

ccNumber

The Credit Card Number. (Not returned on retrieval.)

cvv

The CVV number. (Not returned on retrieval.)

expirationMonth

The month of CC expiration. (Not returned on retrieval.)

expirationYear

The year of the CC expiration. (Not returned on retrieval.)

address

The billing address. (Not returned on retrieval.)

creditCardId

The ID of the Credit Card assigned by WePay.

creditCardName

The name of the Credit Card assigned by WePay.

state

The current state of the card.

referenceId

The Reference ID of the account the card is assoicated with.
