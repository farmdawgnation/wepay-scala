AccountResponse

case class AccountResponse(accountId: Option[Long] = scala.None, name: Option[String] = scala.None, description: Option[String] = scala.None, referenceId: Option[String] = scala.None, accountUri: Option[String] = scala.None, paymentLimit: Option[Long] = scala.None, gaqDomains: Option[List[String]] = scala.None, pendingBalance: Option[Double] = scala.None, availableBalance: Option[Double] = scala.None, currency: Option[String] = scala.None) extends Product with Serializable

The response type for Account API operations that don't return an Account, or some variation thereof.

accountId

The Account ID of the account being considered.

name

The name of the account being considered.

description

The description attached to the account.

referenceId

The reference ID assigne to the account by your application. Reference IDs must be unique per user per app.

accountUri

The URI to redirect the user to to view their account.

paymentLimit

The maximum individual payment that can be made to this account. (deprecated, I think)

gaqDomains

A list of analytics domains assigned to this account, if any.

pendingBalance

The amount on the account that is pending.

availableBalance

The amount on the account that is available.

currency

The currency of the account.
