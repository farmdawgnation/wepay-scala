WePayResourceMeta

trait WePayResourceMeta[Model <: WePayResource[Model]] extends AnyRef

The Meta trait paired with the base WePayResource.

This Meta trait should be used in Meta Objects that are used in the retreival or manipulation of instances of WePay resources. This trait provides functionality for a meta object that is able to run a search based on some search parameters provided in the form of a lift-json object, run some arbitrary query, or retrieve an instance directly by its ID.
