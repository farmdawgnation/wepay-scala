WePayToken

case class WePayToken(userId: Long, accessToken: String, tokenType: String, expiresIn: Option[String]) extends Product with Serializable

An access token that authenticates your application to perform actions on behalf of a certain user on WePay's servers.

userId

The ID of the uer this token relates to on WePay.

accessToken

The actual token.

tokenType

The type of token. Currently, always "BEARER"

expiresIn

An expiration date associated with the token, if any.
